Parli

Learn Fluent

Learn any language

    from any language



Parli is a free, open source language learning app that was created because I wanted to learn languages that either were not on good quality free learning apps or had limited support. Parli offers support for any language on Google Translate, as Google Translate is used to teach. So instead of handcrafted lessons you will get procedually generated lessons that are customisable to your learning needs. You're not forced to learn characters of a different language if you don't want. You can jump straight into building your comprehension and learn enough to converse. Another limitation of more popular language apps is that they usually have very limited support for non-english-speaking learners to learn a language from their own language. But as Parli uses Google Translate as a backend, this problem is no more. Now, anyone can learn any language... from any language.

That being said, Parli does not pretend to be one of the more popular language apps. It will never have the same quality as their lessons. It exists as a way to gain some conversational experience in any given language with focus on accessibility.
